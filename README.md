FileManager

Информация о файлах и директориях выводится при использовании любой команды.
При неккоретности введенной команды, программа выводи информацию об ошибке в информационном окне, а так же записывает код ошибки в файл.

Допускается использование истории введенных команд с применением стрелочек на клавиатуре.

Допускается изменение высоты окна в конфигурационном файле. Данный параметр влияет на количество выводимой информации в окне отображение деревьев файловой системы.
Высота может варьироваться в диапазоне от 13 до 49 пунктов.

Описание команд:
	ls c:\Dir -p 23 - вывод дерева файловой системы с условием "пейджинга".
		ls - Команда
		c:\Dir - Необходимая директория
		-p - Условие "пейджинга" - Необязательное условие
		23 - страница вывода - Необязательное условие
	cd c:\Dir - изменение текущей директории.
		cd - Команда
		c:\Dir - Необходимая директория
	cp c:\Dir c:\CopyDir - копирование каталога/файла.
		cp - Команда
		c:\Dir - копируемая директория/файл
		c:\CopyDir - путь для копирования директории/файла
	rm c:\CopyDir - удаление каталога/файла.
		rm - Команда
		c:\CopyDir - удаляемая диреткория/файл
    file c:\text.txt -p 3 - вывод информации о файле. - В случае, если файл поддерживаемого формата, вывод текстового содержимого файла.
		file - Команда
		c:\text.ru - Необходимый файл
		-p - Условие "пейджинга" - Условие для поддерживаемых тектовых форматов
		3 - Страница вывода - Условие для поддерживаемых текстовых форматов
